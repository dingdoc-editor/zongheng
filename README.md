## zongheng
"纵横" 是钉钉文档表格团队自研的一款基于 JavaScript 的表格 React 组件，主要用于开发 Web 电子表格产品。它是钉钉表格的底层基座，持续稳定支撑着千万级 DAU，有以下特性：
- 基于 Canvas 实现绘制，按需渲染，体验流畅
- 国内一流的公式、数据计算能力与性能
- 开箱即用，拥有丰富的一方插件和组件
- 灵活、方便的扩展能力
- 多人协作友好
- 能力丰富，与Excel兼容性达到95%

`zongheng` 为 "纵横" 的拼音，灵感源于表格由横竖网格交错构成，寓指 "诸子百家，唯我纵横"
